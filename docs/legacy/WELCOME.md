## 🚀 Comece Agora!

### Opção 1: Início Super Rápido (5 minutos)

```bash
npm run install:all
cp backend/.env.example backend/.env
cp frontend/.env.example frontend/.env
createdb gestao_ti
cd backend && npm run seed
npm run dev
```

Acesse: http://localhost:5173

### Opção 2: Guia Passo a Passo

Siga o **[QUICKSTART.md](QUICKSTART.md)** para um guia detalhado.
